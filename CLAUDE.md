# CLAUDE.md

This project is a **MongoDB JSON explorer/curation tool** presented as a desktop/file-manager UI.

If you are starting fresh, read in this order:
1. `README.md` (high-level purpose)
2. `AGENTS.md` (AI-agent quick context)
3. `goal.md` (full workflow + product constraints)
4. `todo.md` (future direction)

Core user workflow:
- connect MongoDB
- browse DBs/collections/documents as folders/files
- quickly sort/label/filter JSON docs for ML data preparation
